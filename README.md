# Colosal Tienda - GitHub Pages

Esta es una página inicial de Colosal Tienda con servidor Node y Express.

## Cómo ejecutar localmente

1. Abre el proyecto en tu terminal.
2. Ejecuta `npm install`.
3. Ejecuta `npm start`.
4. Abre `http://localhost:3000` en tu navegador.

## Desarrollo

- Usa `npm run dev` para iniciar con `nodemon`.
- El servidor sirve los archivos estáticos en la raíz del proyecto.

## Personaliza

- Cambia el número de WhatsApp y los enlaces de Instagram en `index.html`.
- Ajusta la ubicación y los datos bancarios.
- Agrega nuevas secciones como catálogo, promociones o formulario de contacto.
